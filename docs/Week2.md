# Development Log — Week 2

## Objective

Develop an automated approach for locating the optic nerve head (ONH) in retinal en-face OCT images.

## Motivation

The ONH serves as the anatomical reference for nearly every quantitative measurement performed by the pipeline.

Accurate localization enables:

- Standardized retinal orientation
- Automatic quadrant generation
- Consistent RNFL analysis
- Improved reproducibility
- Batch processing
